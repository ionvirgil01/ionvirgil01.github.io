This project is a Python-based tool that uses facial landmark detection and style transfer
to turn normal portraits into stylized cartoon faces.  

**Key features:**
- Built with OpenCV and TensorFlow
- Supports custom artistic filters
- Can run locally or via web app  

You can view the source code and examples on [GitHub](http://github.com).
